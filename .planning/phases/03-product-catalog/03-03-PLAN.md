    Create src/components/catalog/CatalogClient.tsx with 'use client'.

    Imports:
    - useState, useMemo from 'react'
    - Product from '@/lib/types/product'
    - CatalogProductCard from './CatalogProductCard'
    - FilterBar, Category, SortOrder from './FilterBar'
    - FilterDrawer from './FilterDrawer'
    - SearchInput from './SearchInput'

    Props: interface CatalogClientProps { products: Product[] }

    State:
    - activeCategory: Category = 'All'
    - sortOrder: SortOrder = 'latest'
    - searchQuery: string = ''
    - isFilterOpen: boolean = false

    filteredProducts (useMemo, deps: [products, activeCategory, sortOrder, searchQuery]):
    1. Start with [...products]
    2. If searchQuery.trim() is non-empty:
       - Filter by p.name.toLowerCase().includes(searchQuery.toLowerCase())
       - Do NOT apply activeCategory filter (search takes priority)
    3. Else if activeCategory !== 'All':
       - Filter by p.material === activeCategory
    4. Sort:
       - 'price_asc': sort by price_min ascending
       - 'price_desc': sort by price_min descending
       - 'latest': sort by new Date(created_at).getTime() descending
    5. Return result

    showEmptyMessage: boolean = searchQuery.trim().length > 0 && filteredProducts.length === 0

    Layout (outer):
    - Wrapper: <div className="bg-cream min-h-screen">
    - Inner max-width container: <div className="max-w-6xl mx-auto px-4 py-12">
    - Page heading: <h1 className="font-display text-4xl md:text-5xl text-cocoa mb-2">The Collection</h1>
    - Subheading: <p className="font-body text-charcoal/60 mb-8">Premium loc bead accessories, handcrafted in Nigeria.</p>

    Controls bar (flex, gap-3, flex-wrap, mb-8, items-center):
    - SearchInput (value={searchQuery} onChange={setSearchQuery}) — grows to fill available space: className="flex-1 min-w-[200px]"
    - FilterBar (desktop only): hidden md:flex items-center gap-3 — passes activeCategory, onCategoryChange, sortOrder, onSortChange
    - "Filters" button (mobile only): md:hidden — bg-stone border border-charcoal/20 rounded-lg px-4 py-2 text-sm font-body text-charcoal — onClick={() => setIsFilterOpen(true)}

    Empty state message (renders when showEmptyMessage is true, above the grid):
    <p className="font-body text-charcoal/60 text-center py-8">
      No products match &ldquo;{searchQuery}&rdquo; — showing all products
    </p>
    When showEmptyMessage is true, render ALL products (not filteredProducts) in the grid below

    Product grid:
    <div className="grid grid-cols-1 md:grid-cols-3 gap-6">
      {(showEmptyMessage ? products : filteredProducts).map(p => (
        <CatalogProductCard key={p.id} product={p} />
      ))}
    </div>

    FilterDrawer (always rendered, controlled by isFilterOpen):
    <FilterDrawer
      isOpen={isFilterOpen}
      onClose={() => setIsFilterOpen(false)}
      activeCategory={activeCategory}
      onCategoryChange={(cat) => { setActiveCategory(cat); setIsFilterOpen(false) }}
      sortOrder={sortOrder}
      onSortChange={(sort) => { setSortOrder(sort); setIsFilterOpen(false) }}
    />

    When searchQuery changes to non-empty, the category filter is visually bypassed in useMemo — no need to call setActiveCategory('All'). FilterBar chips can appear in their current state; the search simply takes over.
