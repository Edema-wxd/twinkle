    `'use client'`. Props: `{ tags: string[]; activeTag: string | null }`. Use `useRouter` + `useSearchParams` from `next/navigation`. Do NOT use plain `<a>` tags — must use router.push to avoid full page reload (pitfall 3 from research).

    2. Tags query — get all distinct tags: `createClient().from('blog_posts').select('tag').eq('published', true)` then build `[...new Set(rows.map(r => r.tag).filter(Boolean))]`

    Use `createClient()` (public client — public RLS allows reading published posts).

      <BlogCategoryFilter tags={tags} activeTag={category ?? null} />

    **src/components/blog/BlogShareButtons.tsx**
    `'use client'`. Props: `{ title: string; canonicalUrl: string }` — URL passed as prop from Server Component parent (avoids window.location.href SSR crash, see pitfall 2 in research).

    Build share URLs:
    ```typescript
    const waUrl = `https://wa.me/?text=${encodeURIComponent(title + ' ' + canonicalUrl)}`
    const twitterUrl = `https://twitter.com/intent/tweet?text=${encodeURIComponent(title)}&url=${encodeURIComponent(canonicalUrl)}`
    ```
    WhatsApp share URL uses `wa.me/?text=` (no phone number — opens contact picker, distinct from the site WhatsApp CTA which dials Unoma's number).
    Render two `<a>` tags: "Share on WhatsApp" + "Share on X / Twitter". Both `target="_blank" rel="noopener noreferrer"`. Style: small outlined pill buttons.

    **src/app/blog/[slug]/page.tsx**
    Server Component. Next.js 15 async params: `const { slug } = await params`. Also await searchParams if needed (not required here).

    Two Supabase queries:
    1. Post: `createClient().from('blog_posts').select('*').eq('slug', slug).eq('published', true).single()`. If error or no data: `notFound()` — draft posts are 404 on the public page (not accessible, not a valid pending state unlike orders).
    2. Related posts (only if post.tag is non-null): `.from('blog_posts').select('id, title, slug, excerpt, featured_image, tag, published_at').eq('published', true).eq('tag', post.tag).neq('id', post.id).order('published_at', { ascending: false }).limit(3)`. If post.tag is null, skip this query (set relatedPosts to []).

    Construct canonical URL server-side: `const canonicalUrl = \`${process.env.NEXT_PUBLIC_SITE_URL ?? 'https://twinklelocs.com'}/blog/${post.slug}\``

    Page layout:
    ```
    <main className="max-w-3xl mx-auto px-4 py-12">
      {/* Back link */}
      <Link href="/blog" className="...">← Back to Blog</Link>

      {/* Post header */}
      <h1 className="font-heading text-4xl font-bold text-charcoal mt-6 mb-3">{post.title}</h1>
      {post.tag && <span className="...">{post.tag}</span>}
      <p className="text-charcoal/50 text-sm mb-8">{formattedDate}</p>

      {/* Featured image */}
      {post.featured_image && (
        <div className="relative w-full aspect-video mb-8 rounded-lg overflow-hidden">
          <Image src={post.featured_image} alt={post.title} fill className="object-cover" />
        </div>
      )}

      {/* Body — Tiptap HTML */}
      <div
        className="[&_h2]:font-heading [&_h2]:text-2xl [&_h2]:font-bold [&_h2]:mb-4 [&_h3]:font-heading [&_h3]:text-xl [&_h3]:font-semibold [&_h3]:mb-3 [&_p]:mb-4 [&_p]:leading-relaxed [&_p]:text-charcoal/80 [&_ul]:list-disc [&_ul]:pl-6 [&_ul]:mb-4 [&_ol]:list-decimal [&_ol]:pl-6 [&_ol]:mb-4 [&_a]:text-gold [&_a]:underline [&_a]:hover:text-gold/80 [&_strong]:font-semibold [&_em]:italic"
        dangerouslySetInnerHTML={{ __html: post.body }}
      />

      {/* Share buttons */}
      <div className="mt-12 pt-8 border-t border-stone-200">
        <BlogShareButtons title={post.title} canonicalUrl={canonicalUrl} />
      </div>

      {/* Related posts */}
      {relatedPosts.length > 0 && (
        <section className="mt-16">
          <h2 className="font-heading text-2xl font-bold text-charcoal mb-6">Related Posts</h2>
          <div className="grid grid-cols-1 md:grid-cols-3 gap-4">
            {relatedPosts.map(p => <BlogPostCard key={p.id} post={p} />)}
          </div>
        </section>
      )}
    </main>
    ```

    Do NOT use `prose` class — @tailwindcss/typography not installed. Use `[&_tag]:` selectors as shown above.

    BlogShareButtons is a `use client` leaf island. BlogPostCard from Plan 06 Task 1 is a Server Component — import it directly.
