    In src/lib/types/product.ts, add the PriceTier interface before ProductVariant and add price_tiers to ProductVariant:

    ```typescript
    export interface PriceTier {
      qty: number;   // pack size in number of beads (e.g. 25, 50, 100, 150, 200)
      price: number; // price in kobo-less Naira for this pack size
    }

    export interface ProductVariant {
      id: string;
      name: string;
      price: number;              // lowest available tier price (display/sort price)
      in_stock: boolean;
      price_tiers: PriceTier[];   // all available pack sizes with their prices
    }
    ```

    Keep ProductMaterial and Product unchanged. price_tiers is required (not optional) — every variant
    must declare its tiers. Shears uses a single tier [{qty:1,price:3500}].
  </action>
