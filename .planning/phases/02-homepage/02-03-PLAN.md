```
interface ProductCardProps {
  product: MockProduct
  onAddToCart: () => void
}
```

Structure:
- `<article>` element with: bg-stone rounded-xl overflow-hidden hover:-translate-y-1 hover:shadow-lg transition-all duration-200 cursor-pointer
- Image area: `<div className="relative aspect-square w-full">` containing `<Image>` from 'next/image' with props: src={product.image}, alt={product.name}, fill, sizes="(max-width: 640px) 100vw, (max-width: 1024px) 50vw, 25vw", className="object-cover". Use fill NOT width/height.
- Content area: `<div className="p-4">`
  - H3: product.name — font-heading font-semibold text-cocoa mb-1 text-base
  - Price range paragraph: ₦{product.price_min.toLocaleString()} – ₦{product.price_max.toLocaleString()} — font-body text-sm text-charcoal/60 mb-3
  - "Add to cart" button: onClick={onAddToCart}. Styling: w-full bg-gold text-cocoa font-heading font-semibold py-2.5 rounded-lg hover:bg-terracotta hover:text-cream transition-colors text-sm

Imports: Image from 'next/image', MockProduct type from '@/lib/mock/products'

Note: onAddToCart is passed as a prop from the parent Client Component (FeaturedProductsSection). ProductCard itself does NOT need 'use client' — the onClick handler is passed as a prop. However, because ProductCard receives a function prop (onClick), Next.js requires either the component or its parent to be a Client Component. Since FeaturedProductsSection ('use client') renders ProductCard, ProductCard can be either. Make it a Server Component (no 'use client') — Next.js allows Client Components to render Server Components passed as JSX, but here it's simpler: ProductCard is rendered inside a Client Component, so it becomes part of the client bundle. No 'use client' needed on ProductCard itself.

useEffect 1 — Escape key handler:
```
useEffect(() => {
  const handler = (e: KeyboardEvent) => { if (e.key === 'Escape') onClose() }
  document.addEventListener('keydown', handler)
  return () => document.removeEventListener('keydown', handler)
}, [onClose])
```

```
useEffect(() => {
  document.body.style.overflow = 'hidden'

}, [])
```
