  <action>
Create `src/lib/cart/types.ts`:
- Export `CartItem` interface with fields: productId, variantId, tierQty, threadColour (string — empty string for Tools), productName, variantName, unitPrice (Naira), imageUrl, isTool (boolean), quantity (1–10).
- Export `CartState` interface: `{ items: CartItem[]; isDrawerOpen: boolean }`.
- Export `CartAction` as a discriminated union: HYDRATE (payload: CartItem[]), ADD_ITEM (payload: CartItem), UPDATE_QTY (payload: { key: string; qty: number }), REMOVE_ITEM (payload: { key: string }), CLEAR_CART (no payload), OPEN_DRAWER (no payload), CLOSE_DRAWER (no payload).

Create `src/lib/cart/cartReducer.ts`:
- Export `lineKey(item: Pick<CartItem, 'productId'|'variantId'|'tierQty'|'threadColour'>): string` — returns `${productId}:${variantId}:${tierQty}:${threadColour}`.
- Export `initialCartState: CartState = { items: [], isDrawerOpen: false }`.
- Export `cartReducer(state: CartState, action: CartAction): CartState` — pure function:
  - HYDRATE: return `{ ...state, items: payload }` — replaces items only (not drawer state)
  - ADD_ITEM: find existing item by lineKey; if found, update quantity by +1 up to max 10; if not found, append. Also set isDrawerOpen to true.
  - UPDATE_QTY: find item by key; if qty <= 0, remove item; else update qty
  - REMOVE_ITEM: filter out item by key
  - CLEAR_CART: return `{ ...state, items: [] }`
  - OPEN_DRAWER: return `{ ...state, isDrawerOpen: true }`
  - CLOSE_DRAWER: return `{ ...state, isDrawerOpen: false }`

Create `src/lib/cart/threadColours.ts`:
- Export `THREAD_COLOURS` as a readonly const array of objects: `{ id, label, hex }` for the five locked colours: red (#C0392B), black (#1A1A1A), light-brown (#A0785A), blonde (#D4A853), dark-brown (#5C3317). The id values must use hyphens exactly as shown.
- Export `ThreadColourId = typeof THREAD_COLOURS[number]['id']`.
  </action>

Run: `npx tsc --noEmit`
No TypeScript errors. All three files exist and compile cleanly.
