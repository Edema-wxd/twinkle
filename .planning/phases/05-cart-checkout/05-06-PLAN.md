  <action>
Create directory `src/app/api/webhooks/paystack/`. Create `src/app/api/webhooks/paystack/route.ts`.

Do NOT use `'use client'` — this is a server-only route handler.

Imports:
- `import crypto from 'crypto'`
- `import { NextRequest, NextResponse } from 'next/server'`
- `import { createClient } from '@supabase/supabase-js'`
- `import { Database, OrderInsert, OrderItemInsert } from '@/types/supabase'`

Define inline TypeScript interfaces for the Paystack webhook payload shapes:
```typescript
interface PaystackCustomer { email: string }
interface PaystackChargeData {
  reference: string
  amount: number  // in kobo
  customer: PaystackCustomer
  ip_address: string | null
  metadata: PaystackMetadata
}
interface PaystackMetadata {
  cart_items: Array<{
    productId: string; productName: string; variantId: string; variantName: string
    tierQty: number; threadColour: string; unitPrice: number; quantity: number; isTool: boolean
  }>
  customer_details: {
    first_name: string; last_name: string; phone: string
    delivery_address: string; state: string
  }
  subtotal: number
  shipping_cost: number
}
interface PaystackEvent { event: string; data: PaystackChargeData }
```

Export `async function POST(req: NextRequest)`:
1. `const body = await req.text()` — raw body MUST be called before any JSON parsing
2. Get signature: `req.headers.get('x-paystack-signature')` — return 401 if missing
3. Compute HMAC: `crypto.createHmac('sha512', process.env.PAYSTACK_SECRET_KEY!).update(body).digest('hex')` — return 401 if mismatch
4. Parse: `const event: PaystackEvent = JSON.parse(body)`
5. Return 200 immediately after parsing — `NextResponse.json({ received: true }, { status: 200 })`
6. If event.event === 'charge.success', call handleChargeSuccess(event.data) after returning (use void — fire and forget, or await before return — see note below)

NOTE: Actually await handleChargeSuccess before returning to ensure the insert completes. Paystack allows up to 10 seconds. The function should be fast (2 DB inserts).

`async function handleChargeSuccess(data: PaystackChargeData)`:
- Create service-role Supabase client: `createClient<Database>(process.env.NEXT_PUBLIC_SUPABASE_URL!, process.env.SUPABASE_SERVICE_ROLE_SECRET!, { auth: { persistSession: false, autoRefreshToken: false, detectSessionInUrl: false } })`
- Idempotency guard: query `orders` by `paystack_reference = data.reference`; if row exists, return early
- Build OrderInsert object from data and metadata
- Insert into `orders` table and get the inserted `id`
- If insert fails (error || !order), return
- Map metadata.cart_items to OrderItemInsert array (each item: order_id, product_id, product_name, variant_id, variant_name, tier_qty, thread_colour (null if isTool), unit_price, quantity, line_total = unitPrice * quantity)
- Insert all order_items in a single insert call

  </verify>
  <done>
Webhook route exists. Invalid signature returns 401. charge.success event with valid signature inserts into orders and order_items. Duplicate webhook is ignored. TypeScript compiles.
