Create `src/components/checkout/PaystackButton.tsx` with `'use client'` directive:

Props interface:
```typescript
interface PaystackConfig {
  email: string; amountKobo: number; reference: string; metadata: object
}
interface PaystackButtonProps {
  config: PaystackConfig; onSuccess: (reference: string) => void; onClose: () => void; disabled?: boolean
}
```

`handlePay` async function:
- Dynamically import: `const PaystackPop = (await import('@paystack/inline-js')).default`
- `const popup = new PaystackPop()`
- `popup.newTransaction({ key: process.env.NEXT_PUBLIC_PAYSTACK_PUBLIC_KEY!, email, amount: amountKobo, ref: reference, currency: 'NGN', metadata, onSuccess(tx) { onSuccess(tx.reference) }, onCancel() { onClose() } })`

Render: `<button onClick={handlePay} disabled={disabled} className="w-full bg-gold text-cocoa font-heading font-semibold py-4 rounded-lg hover:bg-terracotta hover:text-cream transition-colors disabled:opacity-50 disabled:cursor-not-allowed">Pay ₦{(config.amountKobo / 100).toLocaleString()}</button>`

Create `src/components/checkout/OrderReview.tsx` with `'use client'` directive:

Props: `{ items: CartItem[]; customerDetails: CustomerDetails; onBack: () => void; onPaymentSuccess: (reference: string) => void }`

Compute: subtotal = sum(unitPrice * quantity), shippingCost = getShippingCost(customerDetails.state), total = subtotal + shippingCost, totalKobo = total * 100.

Generate payment reference once on mount: `const [reference] = useState(() => 'TW-' + Date.now() + '-' + Math.random().toString(36).slice(2, 7).toUpperCase())`

Error state: `const [paymentError, setPaymentError] = useState<string | null>(null)`

Layout: heading "Review Your Order" (font-display text-2xl text-cocoa mb-6)

Sections:
1. Line items: compact list showing product name, size, pack, thread colour (if not tool), qty × unit price = line total. `font-body text-sm`. No thumbnails — keep this page light.
2. Divider
3. Price breakdown table: Subtotal, Shipping (with state name in parens — e.g. "Shipping (Lagos)"), Total. Total in font-heading font-semibold text-xl text-cocoa.
4. Customer details summary: name, email, phone, delivery address (read-only, compact, font-body text-sm text-charcoal/70).
5. If paymentError is set: terracotta error banner "Payment was not completed — please try again."
6. Button row: "Back" button (outline style, calls onBack) + PaystackButton

Build the PaystackButton metadata object (strip imageUrl to stay within ~5KB limit):
```typescript
const metadata = {
  cart_items: items.map(({ productId, productName, variantId, variantName, tierQty, threadColour, unitPrice, quantity, isTool }) => ({ productId, productName, variantId, variantName, tierQty, threadColour, unitPrice, quantity, isTool })),
  customer_details: { first_name: customerDetails.firstName, last_name: customerDetails.lastName, phone: customerDetails.phone, delivery_address: customerDetails.deliveryAddress, state: customerDetails.state },
  subtotal, shipping_cost: shippingCost,
}
```

PaystackButton props: config = { email: customerDetails.email, amountKobo: totalKobo, reference, metadata }, onSuccess = onPaymentSuccess, onClose = () => setPaymentError('Payment was not completed — please try again.')
